# Simplificador de expresiones booleanas

Aplicación web para simplificar expresiones booleanas de forma visual e interactiva.  
El proyecto está publicado con GitHub Pages y permite trabajar con expresiones lógicas desde el navegador.

## Demo

Puedes probar la aplicación aquí:

👉 [https://jaimevga.github.io/simplificador-de_expresiones_booleanas.github.io/](https://jaimevga.github.io/simplificador-de_expresiones_booleanas.github.io/)

## Descripción

Este repositorio contiene una página web desarrollada en **JavaScript**, **HTML** y **CSS** que permite ingresar expresiones booleanas y obtener una versión simplificada.

Es un proyecto pensado para facilitar el análisis de lógica booleana, ya sea con fines educativos, académicos o de práctica.

## Características

- Interfaz web sencilla y fácil de usar
- Simplificación de expresiones booleanas
- Diseño responsivo
- Ejecución directa en el navegador
- Publicado mediante GitHub Pages

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Instalación y uso

### Opción 1: usar online
Solo abre la demo publicada en GitHub Pages desde cualquier navegador.

### Opción 2: ejecutar localmente
1. Clona el repositorio:
   ```bash
   git clone https://github.com/JaimeVGA/simplificador-de_expresiones_booleanas.github.io.git
   ```

2. Entra a la carpeta del proyecto:
   ```bash
   cd simplificador-de_expresiones_booleanas.github.io
   ```

3. Abre `index.html` en tu navegador favorito.

Si prefieres, también puedes usar una extensión como **Live Server** en VS Code.
